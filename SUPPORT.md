# Support

## Questions and Help

- Use GitHub Issues for bugs and feature requests.
- Use Discussions (if enabled) for usage questions and ideas.

## What Maintainers Can Help With

- Reproducible bugs in this repository
- Documentation gaps
- Deployment script and configuration issues

## Out of Scope

- Managing your private Discord/GCP credentials
- Custom game rule adjudication for specific chronicles
- Emergency/real-time operational support
