# Security Policy

## Supported Versions

Security updates are applied to the latest `main` branch.

## Reporting a Vulnerability

Please do **not** open public GitHub issues for security vulnerabilities.

Report vulnerabilities privately by:

- Opening a GitHub security advisory (preferred), or
- Emailing: **jasonkennedy@users.noreply.github.com**

Include:

- Affected endpoint/module
- Reproduction steps
- Impact assessment
- Suggested remediation (if available)

We will acknowledge receipt as quickly as possible and coordinate a fix and
disclosure timeline.

## Scope Notes

This app integrates with Discord OAuth, Google Cloud Run, and Google Sheets.
Security reports involving misconfiguration of those services are in scope
when they impact this repository's deployment guidance or defaults.
